## Installation
- `brew install neovim`
- `cd ~/.config`
- `git clone git@github.com:priyanshunayan/nvim.git`
- `run nvim <filename>`

## FAQ:
> fzf is not a valid executable

Run `brew install fzf` and restart nvim

> Fd is not installed

Run `brew install fd` and restart nvim
